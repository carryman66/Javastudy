# 零钱通项目SmallChange
项目实现记账功能，使用java在doc命令窗口实现功能

## 项目需求
- 收益入账
- 消费
- 查看明细
- 退出系统
 
## 任务设计
面向过程，分别实现界面,零钱通明细，收益入账，消费，退出程序

**界面**

1.主方法

2.菜单界面

3.显示菜单，并且可以选择菜单，给出相应提示

**零钱通明细**

1.显示每条入账和消费记录

2.收入和消费都拼接到这里

**收益入账**

1.记录总收入

2.记录当前单笔收入记录

**消费**

1.总收入=总收入-当前收入

2.记录当前单笔消费记录

**退出**

1.循环退出 只能输入y或者n否则就循环，输入n就返回菜单

2.输入y退出

**注意**：一个代码块完成一个功能，一个循环完成一个功能，不要混着写
